# Cost-Estimation软件项目成本估算工具

## 包含

> 功能点估算器

Function Point Estimator v1.0

> COCOMO II估算器

COCOMO II Estimator v1.0

## 作用
能够快速方便的对软件项目进行成本估算，采用可视化的操作，简单易操作。

## 建议

要使用工作簿推荐安装Microsoft office 2007及以上版本，要正常使用请开启宏指令。
具体使用方法请参考用户手册**UserManual.pdf**里面的内容
